# QuestCraft-Minecraft-Plugin
QuestCraft - Minecraft Server Plugin - 1.21.X
A minecraft server plugin providing daily quest for the players to complete to earn 'credits' to buy building items

Are you tired of the 2 week phases? Don't want to gather resources manually? Get QuestCraft for more enjoyable and engaging minecraft server experience to keep your server alive!

QuestCraft adds a simple daily quest system to your Minecraft server. Every day, each player gets 5 new quests featuring tasks like killing mobs, collecting resources, mining ores, and placing blocks.

Complete quests to earn credits, then spend those credits in the quest shop to buy building blocks and items.

---

Quick Commands:

Use /quest help to explore all plugin commands
Use /quest shop to access the quest shop
Check your credit balance by viewing the playerlist (pressing Tab)
Perfect for keeping players engaged with daily goals and rewarding active play. Works on Purpur or Paper1.21.X servers. 

Download the QuestCraft .jar file from the downloads section on this page.
Stop your server. Add plugins while the server is offline to avoid file locks or corruption.
Copy the .jar file into the "plugins" folder of your server directory.
Restart the server and join the server.
You should be able to see the QuestCraft message showing up in your chat box
